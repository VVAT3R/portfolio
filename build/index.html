Config

GPU Mode

BIOS MUX switch (F2 → Configuration → Graphics Mode). Discrete for gaming (full 130W), Hybrid for battery life, iGPU-only for travel. HDMI 2.1 is hardwired to the RTX 3060.

Fan Control

LenovoLegionLinux via AUR — 10-point fan curve GUI, 3 power modes (Quiet/Balanced/Performance), Fn+Q switching. Persists via systemd service.

Power Management

amd-pstate EPP driver (active by default) + TLP. Battery conservation mode caps at 60–80% charge. KDE Power Profiles for quick mode switching.

Gaming Stack

Steam + Proton-CachyOS (Anti-Lag 2 + FSR4 ML), MangoHud overlay, GameMode, Gamescope for HDR. RTX 3060 delivers ~5–20% lower FPS vs Windows depending on the title.

Resizable BAR

Enabled in BIOS — Above 4G Decoding + Re-Size BAR Support. Gives 0–12% FPS gain in supported titles. Confirmed working on linux-cachyos.
